## Overview

This lab demonstrates understanding of JavaScript arrays and related
methods.

All 17 required questions from the assignment have been completed and
clearly labeled in the console output as:

Question 1\
Question 2\
...\

Each question displays:

-   The full array (when applicable)
-   The modified array (when required)
-   The specific result requested
-   Proper formatting as described in the assignment instructions

------------------------------------------------------------------------

## Concepts Covered

This project demonstrates the following JavaScript concepts:

-   Array creation (Literal & Constructor method)
-   Accessing array elements by index
-   Adding elements using `splice()`
-   Removing elements using `delete`
-   Looping using:
    -   `for/in`
    -   `for/of`
-   Sorting arrays
-   Reversing arrays
-   Concatenating arrays using `concat()`
-   Retrieving elements using:
    -   `slice()`
    -   `at()`
-   Multi-dimensional arrays (2D arrays)
-   Filtering using `filter()`
-   Anonymous functions
-   Utility functions:
    -   Removing falsy values
    -   Returning a random item
    -   Finding the largest number in an array
